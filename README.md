# A Projekt Indítása

NodeJS szükséges a projekt futtatásához: [NodeJS](https://nodejs.org/)

1. **Navigálj a frontend könyvtárba:**

   ```bash
   cd evoHike.Frontend
   ```
2. **Telepítsd a szükséges csomagokat** (ezt csak az első alkalommal kell futtatni):

   ```bash
   npm install
   ```
3. **Indítsd el a teljes alkalmazást:**

   ```bash
   npm run start:dev
   ```

Ez a parancs egyszerre elindítja a C# backend API-t és a Vite frontend fejlesztői szervert. A Vite automatikusan meg fog nyitni egy böngészőablakot a frontendhez.
